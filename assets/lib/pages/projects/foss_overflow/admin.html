Admin

The Admin has the following features:
  1. Add and Manage Students

    2. The admin has the option to add, view, delete students. Through this feature the admin can add students which can access the app for their respective institute.

    Student Entry Multi Choice Roles
    Graduation Year Picker View Students


    The add student has probably the most diverse set of input options where we have a year picker for picking the year of graduation, a multi-choice role picker for extra roles like president, class rep, etc.

  2. Add and Manage Courses

    3. The admin has the option to add, view, delete courses.

    Course Entry View Courses


    The course entry has a single choice picker for selecting the primary room corresponding to the course and also a multi-choice picker for selecting the branches that the course is being taught in.

  3. Add and Manage faculties

    4. The admin has the option to add, view, delete faculty.

    Faculty Entry View Faculties


    Faculty entry has a multiple choice picker for selecting the courses that the faculty is teaching. These courses are the same courses that the admin has added in the course entry section.

  4. Add and Manage Mess Menus

    5. The admin has the option to add, view, delete mess menus.

    Menu Entry Manage Menu


    Both the menu entry and view menu pages feature a unique and easy to use UI for adding and viewing mess menu items.

  5. Add and Manage Rooms

    6. The admin has the option to add, view, delete rooms.

    Manage rooms



    The manage rooms page has a simple grid like UI for adding and viewing rooms same as the home page.
