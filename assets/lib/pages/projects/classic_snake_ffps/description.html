Classic Snake Game in C++

The Classic Snake Game in C++ is a simple yet addictive game that runs in the command line interface. It offers a nostalgic experience for players, bringing back memories of the early days of computer gaming. Despite its simple text-based nature, the game provides hours of entertainment and challenges players to beat their own high scores.

  • Pixels updated only when needed:

    • The game follows a similar method as the 'Dirty Region', where instead of rendering the entire frame, the parts of frames are updated where the objets are updated

  • No Graphics, Pure Command Line Game:

    • The game is entirely text-based and runs in the command line interface. Players control the snake using the arrow keys to navigate through the grid.

  • Overlay Scoreboard:

    • The game features an overlay scoreboard that displays the player's current score and highest score achieved in the current session. Players can aim to beat their own records and compete with friends for the top spot on the scoreboard.

  • Customizability Options:

    • The game offers various customizability options to make the experience unique for each player. Players can customize the following aspects of the game:

    • Snake Head Shape:

      • Players can choose from different shapes for the snake's head, such as square, triangle, or circle.

    • Snake Head Color:

      • Customize the color of the snake's head to make it stand out on the grid.

    • Food Shape:

      • Players can select the shape of the food that the snake eats, such as square or circle.

    • Food Shape Color:

      • Customize the color of the food shape to make it visually appealing.

    • Body Shape:

      • Choose from various shapes for the snake's body segments, like rectangle or oval.

    • Body Color:

      • Customize the color of the snake's body to make it distinct from the background.

  • File Handling for Customizability Settings:

    • The game uses file handling to save the customizability settings in binary format. This way, players can have their preferred settings every time they launch the game, even after closing it.
