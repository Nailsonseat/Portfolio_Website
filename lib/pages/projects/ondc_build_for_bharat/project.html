Landing Page

The landing page is a simple appbar with two buttons and add product and metrics. A simple grid builder to display the added products



Add Product

The add product page is divided into three parts, basically inputs for text, image, audio respectively



Loading Page

The loading page features amazing asynchronous magic behind the screens, I think this is the most amount of async and Future related code I have ever written.



Product Edit/Confirmation Page

As the generated content is prone to errors therefore we introduced a edit product page to correct and edit the product details before adding it to the list of products in the landing page.



Metrics

The metrics page is a simple page describing the average time taken to each input to process.
