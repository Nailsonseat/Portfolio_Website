Features in detail

  • Context Generation

    There are times when you encounter a question during your test or exam that is unimpressive, stale, and devoid of any creativity. This kind of question not only looks bad but their lack of context and relatability doesn't help the students either. Therefore Context Generation aims to introduce context to the question using GPT models at a click of a button. Thus questions with context will make the assessment process that much more enjoyable and help the students to better implement the learned concepts in the real world.

  • Template Randomizer

    In a question the user has the option to mark variables in a question then they have the option to mark some variables, so that when the question is published to students, the text marked as variables are randomized. There are filters that can be applied on a variable so that the randomness of a variable is in control of the user.

  • Question segmentation

    Well now that we have introduced context in our questions, it is apparent that the length of questions has been significantly increased, that is going to be very hectic for the students to read and interpret. Hence the questions can also be segmented into easy to digest smaller parts.

  • Question distribution

    The teachers can create their own classrooms, they can distribute the questions in form of assignments, tests, exams etc.
