Below is the statement for milestone 1

Focus: Identify User Requirements

Envision different experiences/scenarios which you have encountered while interacting with the SEEK portal. Create a learner journey map of these scenarios (without GenAI).

Envision how the learner journey map will change if GenAI is integrated into each of these scenarios. Create a modified learner journey map for each of these scenarios.Think of all possible integrations.You can then choose which ones to take forward.

Write user stories based on the learner journey maps. The user stories should be based on the SMART guidelines discussed in the lectures

The user stories should be in the following format:

As a [type of user],

I want [an action],

So that [a benefit/value]
