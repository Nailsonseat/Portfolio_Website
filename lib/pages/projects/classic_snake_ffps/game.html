A brand new Menu

The original version didn't even have a menu. This is a brand new menu that I have created. It is a simple menu with a few options. The menu is as follows:
  1. Start

    2. This is the main option. It is the option that the user will use to play the game. When the user clicks on this option, the game will start.

  2. Options

    3. This is the option that the user will use to change the settings of the game. When the user clicks on this option, the game will take the user to the options menu.

  3. Exit

    4. This is the option that the user will use to quit the game. When the user clicks on this option, the game will quit.





Options Section

The options section is where the user can change the appearance of the snake and the food. All of these options are created using Unicode characters. There are six appearance options for the snake and the food. The options are as follows:
  • 2 options for snake's body
  • 2 options for snake's head
  • 4 options for food
  • 4 options for food color
  • 7 options for snake's head color
  • 7 options for snake's body color




Gameplay

The gameplay is simple. The user controls the snake using the arrow keys or WASD. The snake has to eat the food to grow. The walls are not solid, so the snake can go through the walls and come out of the other side. The game ends when the snake collides with itself.

The coordinates of the snake and the food are stored in a list vector and its length is increased everytime the snake eats the food. The upper right corner of the screen shows the score of the player.

The focused frames per second basically means that the game doesn't loop the terminal on which the game is displayed row or column wise, but just jumps to the coordinates straightaway. This is me digging through the old discontinued conio.h header and making some adjustments to use it in the game.
